# Three-Body Simulation

A Three.js-based visualization of the three-body problem, using the 4th-order Runge-Kutta (RK4) method for numerical integration.

## Quick Start

Open `index.html` directly in your browser.

## File Structure

```
three-body-sim/
├── index.html    # Page structure
├── main.js       # Simulation logic
├── style.css     # Styles
```

## Features

### Simulation
- **RK4 Integrator** - 4th-order Runge-Kutta for celestial motion
- **Random Initial Conditions** - Random position/velocity with center of mass at origin
- **Custom Initial Conditions** - User-defined position and velocity
- **Dual Trail Display** - Short trail + long trajectory

### Controls
- **Reset** - Reinitialize simulation
- **Play/Pause** - Start or pause simulation
- **Apply** - Apply current parameters
- **Screenshot** - Save current view as PNG
- **Speed** - 1x ~ 100x
- **Trail Toggle** - Show/hide short trail
- **Path Toggle** - Show/hide long trajectory
- **Random Toggle** - Random or custom initial conditions

### Internationalization
- **Language Toggle** - Chinese/English bilingual support
- **Real-time Distance Display** - Shows distances between all three bodies

### UI/UX
- **Beautiful Animations** - Title shimmer effect, panel slide-in, button hover feedback
- **Hide Cursor on Drag** - Auto-hide cursor when dragging the view

### Parameters
- **Mass M1/M2/M3** - Mass of three bodies (0.1 ~ 100)
- **Initial Position P1/P2/P3** - Position vectors
- **Initial Velocity V1/V2/V3** - Velocity vectors

## Known Characteristics

- Energy drift should remain below 5%
