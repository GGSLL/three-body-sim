# 三体模拟 (Three-Body Simulation)

基于 Three.js 的三体问题可视化模拟，采用四阶龙格-库塔法 (RK4) 进行数值积分。

## 在线预览

直接在浏览器中打开 `index.html` 即可运行。

## 功能特性

### 模拟
- **RK4 积分器** - 四阶龙格-库塔法计算天体运动
- **随机初始条件** - 随机生成位置和速度，保证质心在原点
- **自定义初始条件** - 用户可输入位置和速度
- **双轨迹显示** - 拖尾 (短) + 轨迹 (长)

### 控制
- **重置** - 重新初始化
- **播放/暂停** - 开始或暂停模拟
- **应用** - 应用当前参数
- **截图** - 保存当前画面为 PNG 图片
- **模拟速度** - 1x ~ 100x
- **拖尾开关** - 显示/隐藏短拖尾
- **轨迹开关** - 显示/隐藏长轨迹
- **随机开关** - 随机或自定义初始条件

### 国际化
- **语言切换** - 支持中文/英文双语切换
- **实时距离显示** - 显示三个天体两两之间的距离

### UI/UX
- **精美动画** - 标题流光效果、控制面板滑入、按钮悬停反馈
- **拖拽隐藏光标** - 拖动视角时自动隐藏光标

### 参数
- **质量 M1/M2/M3** - 三个天体的质量 (0.1 ~ 100)
- **初始坐标 P1/P2/P3** - 位置向量
- **初始速度 V1/V2/V3** - 速度向量

## 已知特性

- 三体问题是混沌系统，天体长时间后可能聚集或逃逸
- 能量漂移应保持在 5% 以下

## 技术栈

- **Three.js** - 3D 渲染引擎
- **OrbitControls** - 视角控制
- **原生 JavaScript** - 无需构建工具
