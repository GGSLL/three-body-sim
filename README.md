# 三体模拟

基于 Three.js 的三体问题可视化，采用四阶龙格-库塔（RK4）方法进行数值积分。

## 快速开始

- **在线访问**: https://gmydwz.dpdns.org
- **本地运行**: 直接在浏览器中打开 `index.html`。

## 功能特性

### 模拟
- **RK4 积分器** - 四阶龙格-库塔方法计算天体运动
- **随机初始条件** - 随机位置和速度，质心位于原点
- **自定义初始条件** - 用户自定义位置和速度
- **双轨迹显示** - 短轨迹 + 长轨迹

### 控制
- **重置** - 重新初始化模拟
- **播放/暂停** - 开始或暂停模拟
- **应用** - 应用当前参数
- **截图** - 将当前视图保存为 PNG
- **速度** - 1倍 ~ 100倍
- **轨迹开关** - 显示/隐藏短轨迹
- **路径开关** - 显示/隐藏长轨迹
- **速度开关** - 显示/隐藏速度矢量
- **随机开关** - 随机或自定义初始条件
- **GitHub** - 跳转到项目仓库

### 国际化
- **语言切换** - 中文/英文双语支持
- **实时距离显示** - 显示三体之间的距离

### 用户界面/体验
- **精美动画** - 标题闪烁效果，面板滑入，按钮悬停反馈
- **拖动时隐藏光标** - 拖动视图时自动隐藏光标

### 参数
- **质量 M1/M2/M3** - 三个天体的质量 (0.1 ~ 100)
- **初始位置 P1/P2/P3** - 位置矢量
- **初始速度 V1/V2/V3** - 速度矢量
